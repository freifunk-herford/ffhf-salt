# Freifunk Herford SaltStack Configuration

Anleitung unter <https://freifunk-herford.github.io/ffhf-salt/> lesen.

## Generate Documentation

    cd contrib
    virtualenv venv
    venv/bin/pip install --upgrade pip -r requirements.txt
    bash generate-docs.sh

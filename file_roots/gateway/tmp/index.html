/tmp¶

Wir stellen sicher, dass das temopräre System Verzeichnis tmp besteht und über die korrekten Benutzerrechte verfügt.

Die kann mit den folgenden Befehlen erreicht werden.

mkdir /tmp
chown root:root
chmod 1777

Salt State File¶

tmp.sls

# /tmp

/tmp:
  file.directory:
    - name: /tmp
    - user: root
    - group: root
    - mode: 1777
